# 📦 Sistema de Gerenciamento de Estoque (Console)

Um sistema de gerenciamento de estoque via terminal desenvolvido 100% em Java. Este projeto foi criado com o objetivo de aplicar e consolidar os fundamentos da Programação Orientada a Objetos (POO) e manipulação de coleções de dados na memória.

## 🛠️ Funcionalidades
- **Cadastrar Produto:** Permite inserir novos itens no estoque (Nome, Preço e Quantidade), gerando um ID único automaticamente.
- **Listar Estoque:** Exibe todos os produtos cadastrados de forma formatada.
- **Relatório Financeiro:** Calcula dinamicamente o valor total investido em mercadorias com base no preço e quantidade de cada item.
- **Excluir Produto:** Busca e remove um produto específico através do seu ID.

## 💻 Tecnologias e Conceitos Aplicados
- **Java SE**
- **Orientação a Objetos (POO):** Classes, Objetos, Encapsulamento (Getters/Setters) e modificadores de acesso.
- **Estruturas de Dados:** Uso de `ArrayList` para armazenamento dinâmico.
- **Controle de Fluxo:** Laços de repetição (`while`, `for`, `foreach`) e estruturas condicionais (`switch-case`, `if`).
- **Atributos Estáticos:** Uso de `static` para simular o Auto-Incremento de IDs.

## 🚀 Como executar o projeto
1. Clone este repositório: `git clone https://github.com/Guims0/Sistema1.0.git`
2. Abra o projeto na sua IDE de preferência (IntelliJ, Eclipse, VS Code).
3. Execute a classe `Main.java`.
4. Interaja com o menu diretamente pelo console!
